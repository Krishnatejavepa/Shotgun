# Starter Templates

Pre-filled skeletons for common founder types. During onboarding (Phase 2), if the venture clearly matches one of these, use it as the starting shape for `memory/venture.md`, `memory/metrics.md`, and the vault taxonomy instead of building from a blank page. Replace every `<angle-bracket>` placeholder with the founder's real answer; delete sections that don't apply.

| Template | For |
|---|---|
| `saas.md` | Software products with recurring revenue |
| `creator.md` | Content businesses: newsletters, YouTube, courses, communities |
| `agency.md` | Client-services businesses: agencies, consultancies, freelancers scaling up |
| `ecommerce.md` | Stores selling physical or digital products: Shopify, Etsy, Gumroad, own checkout |

Each template's "numbers that matter" section defines what the weekly pulse tracks in `memory/metrics.md`.

Templates are suggestions, not forms. A founder who doesn't fit any of them gets the plain `memory/_templates/venture.md`. Contributions of new verticals welcome, see CONTRIBUTING.md.
